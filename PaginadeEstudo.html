Conceitos basico de quando é utilizado

Usados quando precisamos repetir alguma coisa um número determinado ou indeterminado de vezes

Laços de repetição (FOR/WHILE/DO-WHILE)

Laço FOR

Usado quando se sabe o número de repetições desejadas

É utilizado quando o número de iterações é conhecido previamente ou para percorrer coleções (arrays, listas). Exemplos incluem imprimir uma contagem (1 a 10), percorrer itens de uma lista de compras, processar todas as notas de uma turma ou gerar anos em um campo de seleção

Clique aqui para saber mais

Laço WHILE

Primeiramente testa a condição da variavel antes de ela entrar no laço.Se nao atender aos requisitos o laço nem é iniciado

É utilizado para repetir um bloco de código enquanto uma condição específica for verdadeira, sendo ideal quando não se sabe antecipadamente quantas iterações serão necessárias. Exemplos comuns incluem ler dados do usuário até que ele insira um valor válido (validação), processar um arquivo até o final ou manter um jogo rodando

Clique aqui para saber mais

Laço DO-WHILE

Primeiro é executado e depois testa a variavel. Executado sempre uma vez pelo menos

É utilizado quando o bloco de código precisa ser executado pelo menos uma vez, independentemente da condição ser verdadeira ou falsa de antemão. É ideal para menus interativos, validação de entrada de dados ou ações que exigem uma execução inicial obrigatória antes da verificação
